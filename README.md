# CodeGuardian - Capstone Project

CodeGuardian is a comprehensive full-stack application designed for secure code analysis and student project management. It features an AI-powered chatbot, automated code similarity detection, and a robust security configuration.

## 🚀 Tech Stack

### Frontend
- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Language:** TypeScript

### Backend
- **Framework:** Spring Boot (Java)
- **Database:** MySQL
- **AI Integration:** Google Gemini API
- **Security:** Spring Security + JWT

## 📂 Project Structure

- `backend/`: Spring Boot application containing business logic and security configurations.
- `frontend/`: React application providing a modern, responsive user interface.
- `docs/`: Project documentation and manuals.

## 🛠️ Getting Started

### Prerequisites
- JDK 17+
- Node.js 18+
- MySQL Server

### Local Development

1. **Backend:**
   ```bash
   cd Backend
   ./mvnw spring-boot:run
   ```

2. **Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
